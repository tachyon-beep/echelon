# Issue Tracking

This directory tracks bugs, enhancements, and jank. Copy the appropriate template and rename it.

## Templates

| Template | Use For |
|----------|---------|
| `TEMPLATE_BUG.md` | Something is broken — expected vs actual behavior |
| `TEMPLATE_ENH.md` | Minor improvement — small feature or UX tweak |
| `TEMPLATE_JANK.md` | Code smell — works but shouldn't stay this way |
| `TEMPLATE_WONTFIX.md` | Intentional behavior — looks wrong but is correct by design |

## Naming Convention

```
BUG_YYYY-MM-DD_short-description.md
ENH_YYYY-MM-DD_short-description.md
JANK_YYYY-MM-DD_short-description.md
WONTFIX_short-description.md
```

## Workflow

1. Copy template
2. Fill in details
3. Set status to Open
4. Update status as work progresses
5. Move to `archive/` when resolved (optional)
